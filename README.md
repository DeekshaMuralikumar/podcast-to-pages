# 📖 Podcast to Pages

**Episodes written instead of recorded.**

Podcast to Pages is a personal website inspired by my Medium series, where every article is presented as an episode. Readers can explore essays by theme, enjoy a book-like reading experience, or listen to AI-generated narration.

## Features

* 📚 Book-inspired reading experience
* 🎧 Audio mode for each episode
* 🏷️ Theme-based organization
* 🔍 Search and filter episodes
* 📱 Responsive design

## Tech Stack

* React JS
* React Router
* HTML
* CSS
* JavaScript

Deployed - Vercel: https://podcast-to-pages.vercel.app/
